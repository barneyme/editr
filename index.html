editr Settings

Open editr.cc in web browser and start writing. Changes are automatically saved in your browser. ("Save" a copy of your file before clearing your browser cache.)
editr can be used offline after the first use. Reopen editr.cc in your browser to continue working on your file.
Files can be opened from your device using the "Open" button or by dragging and dropping in to editr. To save a file outside of your Downloads folder enable the "Ask where to save each file before downloading" setting in your browser.
editr PWA can edit and save local files directly once installed.

CTRL + O (📂 Open) to open a file from your computer into current buffer.
CTRL + S (💾 Save) to save current buffer to your computer (CTRL + SHFT + S to Save As).

CTRL + . ( CSV) to toggle CSV grid.
CTRL + ; to save as .html when in Markdown preview.
CTRL + ' to save as .pdf when in Markdown preview.
CTRL + 1-9 (üìÅ) to switch between file buffers (up to 9 files).
CTRL + G (üÉè Blackjack) to play a game of Blackjack.
CTRL + F (🔍 Find) to find and replace text.
CTRL + K (🔗 Link) to open a web link on current line in a new tab.
CTRL + / (e Settings) to open the editr help and settings window.
CTRL + = (üîí Lock) to lock the current note with a password, or unlock an existing one.
